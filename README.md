# Minecraft Color Code Generator

Proyek ini adalah tool sederhana berbasis web untuk menghasilkan dan melihat preview kode warna serta format teks Minecraft. Tool ini mendukung baik edisi Java maupun Bedrock.

## Features

- **Dark Mode Support**: Beralih antara tema terang dan gelap.
- **Java and Bedrock Support**: codes format Minecraft Java dan Bedrock.
- **Live Preview**: Lihat bagaimana teks yang diformat warna akan terlihat secara real-time.
- **Button Copy to Clipboard**: Salin teks yang diformat dengan cepat untuk digunakan di server atau dunia Minecraft kamu.
